## Teknik Mimari

### Browser Daemon (En İlginç Kısım)
```
İlk komut: ~3 saniye (Chromium başlatma)
Sonraki komutlar: ~100-200ms
```

Long-lived Chromium daemon çalışıyor — her komutta yeni browser açılmıyor. State dosyası (`.gstack/browse.json`) PID, port, auth token saklar.

**Element adresleme** → `@e1`, `@e2` gibi ref'ler Playwright Locator + accessibility tree kullanıyor. DOM mutation değil, CSP ihlali yok, Shadow DOM sorunu yok.

**Güvenlik:**
- Sadece localhost bind
- Bearer token authentication
- Cookie'ler disk'e şifresiz yazılmıyor
- Shell injection için hardcoded registry + parameterized spawning
